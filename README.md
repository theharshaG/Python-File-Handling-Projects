# 1. ATM Process System

## Description
A simple ATM simulation program that allows a user to perform basic banking operations.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Store balance using file handling

## Concepts Used
- File reading and writing
- Menu-driven programs
- Conditional statements
- Loops

# 2. Library Management System

## Description
A small system that manages library books using file storage.

## Features
- Add books
- View book list
- Search books
- Store book records in a file

## Concepts Used
- File handling
- Lists and loops
- Searching logic
- Data storage
 
# 3. Contact Book

## Description
A contact management system that saves and retrieves contacts from a file.

## Features
- Add new contact
- View all contacts
- Search contact
- Delete contact

## Concepts Used
- Python file handling
- Dictionary-like data storage
- Searching records
- Menu-driven interface
